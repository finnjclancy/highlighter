Highlighter

Highlight any page. Share what you read.

A clean browser extension that lets you highlight text on the web in custom colours, organize quotes into folders, and share live links so anyone with the extension sees the same highlights on the page.

Get the extension How it works
What it does

Highlight in any colour

Custom palette with text + background per swatch. Hover any highlight to re-colour or remove it.

Folders, comments, tags

Organize quotes the way you think. Export any folder or selection as plain text or Markdown.

Draw on the page

Pen, line, and rectangle tools for marking up diagrams, screenshots, or PDFs in-place.

Share what you read

A live link recipients open to see your highlights painted onto the original page (extension required for live overlay).

Built with no servers, no tracking. Highlights live in your own browser storage.
